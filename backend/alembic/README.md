# Alembic Migration Environment

## Overview

This directory contains the Alembic migration configuration for QuantX AI.

## Structure

- **env.py** - Migration environment configuration
- **script.py.mako** - Migration script template
- **versions/** - Migration scripts (one per revision)
- **README.md** - This file

## Workflow

### Creating a New Migration

```bash
alembic revision --autogenerate -m "description"
```

### Running Migrations

```bash
# Upgrade to latest
alembic upgrade head

# Downgrade one revision
alembic downgrade -1
```

### Best Practices

- Always review autogenerated migrations before applying
- Never edit migration scripts after they have been applied to production
- Include both `upgrade()` and `downgrade()` functions
- Use `batch_alter_table` for SQLite-compatible migrations when needed
